Week 04 Discussion 02: Deconstructing Web Design

Website Link: Disney.com

One element I inspected was the “Get Tickets Now” button in the Toy Story 5 featured banner. In the CSS rules, the button uses left and right margins (margin-left: 12px; and margin-right: 12px;) to create spacing between buttons and surrounding elements. This margin prevents the buttons from appearing crowded and improves the overall visual organization of the page.

The featured content area also uses padding within the banner section to create space around the text and buttons. The combination of margins and padding helps draw attention to the call-to-action buttons while maintaining a clean and professional appearance.

Disney.com uses Flexbox extensively throughout its homepage. In Developer Tools, I found several containers identified as Flexbox layouts, including span.btn-container, div.cta-items-container, and div.item.entity-box.

These elements use display: flex to arrange content efficiently. For example, the button container aligns the “Get Tickets Now” and “Learn More” buttons within the hero section, while other Flexbox containers organize movie and promotional content cards into responsive layouts. Flexbox allows the website to maintain proper alignment and spacing across different screen sizes.
